Privacy Policy

TempoPass — Chrome Extension
Last updated: March 22, 2026

Overview

TempoPass ("the Extension") is a Chrome browser extension that creates time-limited, spending-capped access keys for Tempo blockchain wallets. This privacy policy explains what data the Extension accesses, how it is used, and how it is stored.

Data We Collect

The Extension does not collect, transmit, or share any personal data with the developer or any third party. All data remains on your device.

Data Stored Locally

The Extension stores the following data in Chrome's built-in storage APIs on your device:

  • Wallet credential reference (chrome.storage.local): Your passkey credential ID and public key, read from wallet.tempo.xyz during the one-time connection step. This is not your private key — your private key never leaves your device's secure hardware enclave (e.g., Secure Enclave on macOS, TPM on Windows).
  • Active access key session (chrome.storage.session): The P256 private key of the currently active access key, its on-chain address, expiry timestamp, and spending limit. This data is automatically cleared when Chrome closes or when the key expires.
  • Transaction log (chrome.storage.local): A record of the last 20 transactions made through the Extension, including transaction hash, amount, recipient address, and timestamp.

Data Accessed from Websites

During the one-time wallet connection flow, the Extension briefly opens wallet.tempo.xyz in a background tab and reads your passkey credential ID and public key from that page's localStorage. The tab is immediately closed. The Extension does not read, modify, or access any other website data.

Content Scripts and MPP

The Extension injects a small content script on all web pages to support the Micropayment Protocol (MPP). This script intercepts HTTP 402 (Payment Required) responses and facilitates automatic payment using your access key. The content script:

  • Does not read or modify page content, DOM elements, cookies, or localStorage of any website other than wallet.tempo.xyz during connection.
  • Does not track your browsing history or activity.
  • Only activates when a website returns an HTTP 402 response with a Payment authentication challenge.

Network Requests

The Extension makes network requests only to:

  • Tempo blockchain RPC endpoint (rpc.presto.tempo.xyz): To submit transactions, check balances, and query access key status. These are standard blockchain RPC calls and do not include any personal identifying information.

The Extension does not communicate with any analytics services, advertising networks, or telemetry endpoints.

Third-Party Services

The Extension does not use any third-party analytics, tracking, or advertising services. The only external service accessed is the Tempo blockchain network via its public RPC endpoint.

Data Sharing

The Extension shares no data with the developer or any third party. All data remains local to your Chrome browser and your device.

Data Deletion

You can delete all Extension data at any time by:

  • Clicking "Disconnect" in the Extension popup, which clears all stored data.
  • Uninstalling the Extension, which removes all associated Chrome storage.

Security

Your passkey private key is never accessed, displayed, or stored by the Extension. Access keys are P256 keypairs generated locally in your browser, authorized on-chain with your passkey, and subject to on-chain spending limits and expiry enforced by the Tempo protocol smart contracts.
